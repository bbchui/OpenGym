# API Endpoints

## HTML API

### Root
* `GET /` - loads React web app

## JSON API

### Users
* `POST /api/users`
* `PATCH /api/users`

### Session
* `POST /api/session`
* `DELETE /api/session`

### Gyms
* `GET /api/gyms`
* `POST /api/gyms`
* `GET /api/gyms/:id`
* `DELETE /api/gyms/:id`
* `PATCH /api/gyms/:id`

### Reviews
* `GET /api/gyms/:id/reviews`
* `POST /api/gyms/:id/reviews`
* `GET /api/gyms/:id/reviews/:id`
* `PATCH /api/gyms/:id/reviews/:id`
* `DELETE /api/gyms/:id/reviews/:id`
